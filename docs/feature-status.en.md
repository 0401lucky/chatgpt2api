## Go 后端迁移状态

当前默认 Docker 镜像已经切到 Go 后端。根目录 `Dockerfile`
会构建原 Next.js 静态网页和 Go API，并在同一个容器内通过
`nginx + chatgpt2api-go` 提供服务。

仍需深迁和压测的部分：

- 注册机核心自动注册执行器已接入 Go，当前优先支持 `gptmail` / `yyds_mail`。
  配置、状态、日志推送、账号补充、账号池指标和 YYDS 域名信誉回写已经迁移；
  其它邮箱 provider 后续继续补齐。
- R2 备份当前只做本地归档，真实远端上传还在后续规划。
- 图床当前支持连接测试，自动上传替换仍需继续深迁。
- `/v1/responses` 与 `/v1/messages` 已接入，但完整流式事件和更复杂的兼容细节还要继续补齐。
- 还没有完成大账号池正式压测。
